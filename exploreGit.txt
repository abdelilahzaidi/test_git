Git :permet de faire des points de sauvegardes.
----
-Installer l'extension git graph
-Quand les sauvegardes sont en locales, elles se font toutes dans le .git
1-Telecharger git à partir du site officiel
2-Initialiser un projet git : git init
3-Créer une branche : git branch nom_branche
4-git branch permet de montrer les branches existantes
